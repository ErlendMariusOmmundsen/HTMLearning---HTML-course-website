Changes to homepage - index.html

A "Start learning" button repaced the intro text in the mockup. This way more experienced users can spend only two clicks to choose the course they want, and beginner users will immediately get an impression of the kind of courses that HTMLearning offers.

Changes to courses overview - courses_overview.html

To keep the design consistent, both the courses box-links on this page, and the lecturers box-links on the about page become bigger on hover. In order to differentiate the course boxes from the navbar, but also identify them as buttons, the color of the boxes are the color of the lightest color of a button on hover.

Changes to FAQ - help.html

In the FAQ when hovering over a question it changes its style to that of a link to indicate that it is clickable. Also instead of sliding down the answer it is now shown immediately.

Changes to login and sign up - login.html

The sizes are changed somewhat to make the page more responsive. The white box is changed to 510 x 370, which changes a bit depending on screen size and zoom. The tabs are now sized by 255 x 64. The sign in and sign up button is now 293 x 37 which changes based on screen size and zoom.

Changes to quiz - quiz.html

The evaluation at the end of the quiz only has two evaluations, can and can't, instead of three.

Had there been more than one quiz, the JSON files would have been put in a folder, and we would have had to find a way to set the "Back to course" button and the right JSON path for the different courses.

Added Javascript file courses.js

In order to share the desired behaviour of displaying and hiding the reviews, and faking a registered for course response across all individual course pages, this file was added.
