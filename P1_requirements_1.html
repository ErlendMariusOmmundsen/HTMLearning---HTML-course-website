Group 1 - requirements draft

by Martin Skurtveit, Joakim Fremstad, Erlend Marius Ommundsen, Lærke Svendsen
Written 18.09.2018

Client

Company which offers HTML-courses: HTMLearning AS Ltd. GmbH.

Purpose and Goals:

The site wishes to inform users about upcoming courses, prices, and other details about the courses.

Audience:

Users interested in learning HTML, but also other web technologies.

Requirements

Functional requirements

  1. Navigation menu
  2. FAQ site
    1. FAQ questions should only display the question, then when it is clicked the answer will be displayed under the question.
  3. Send mail to the company
  4. Overview of all courses

Non-functional requirement

  1. Every page must load in less than 1 second.
  2. The user should be able to navigate the menu from all pages
  3. All text should be readable

Final location

https://www.htmlearning.academy/
